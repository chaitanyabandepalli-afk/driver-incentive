# Abstract

The Driver Incentive & Performance Bonus Tracker is a digital prototype developed for Manivtha Tours & Travels to manage and evaluate driver performance in a structured way. The system focuses on tracking key performance indicators such as punctuality, customer rating, trips completed, and monthly incentive payout.

The main purpose of this project is to replace manual tracking methods like spreadsheets, WhatsApp messages, and paper records with a centralized digital platform. Staff or admin users can enter driver performance data through an entry form, and the system automatically calculates the incentive amount based on predefined business rules. The dashboard allows management to view, search, filter, and monitor all driver records in one place.

The system also includes a detail view, reports, analytics, and payment status tracking. By using this tracker, Manivtha Tours & Travels can reduce calculation errors, improve transparency, reward top-performing drivers fairly, and encourage better service quality.

The final prototype will include frontend screens, backend APIs, database storage, calculation logic, testing documentation, GitHub repository, deployment URL, project report, PPT, and demo video.
