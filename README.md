# Driver Incentive & Performance Bonus Tracker

## Company
Manivtha Tours & Travels

## Project Overview
The Driver Incentive & Performance Bonus Tracker is a digital system designed to help Manivtha Tours & Travels track driver performance and automatically calculate monthly incentive payouts.

The system tracks important driver KPIs such as punctuality, customer rating, trips completed, complaints, incentive amount, and final payout.

## Problem
Currently, driver performance and incentive calculation can become difficult if handled manually through spreadsheets, WhatsApp messages, phone calls, or paper records. Manual tracking can lead to errors, delays, confusion, and lack of transparency between drivers and management.

## Solution
This system provides a centralized digital platform where admin or staff users can enter driver performance data, view records in a dashboard, calculate incentives automatically, and generate reports for management.

## Main Users
- Admin
- Staff
- Management

Drivers are the people whose performance is tracked. They do not directly update dashboard data.

## Main Modules
1. Admin Dashboard
2. Driver Performance Entry Form
3. Driver Incentive Dashboard
4. Detail and History View
5. Reports and Analytics Dashboard
6. Automated Incentive Calculation Engine
7. Testing and Deployment

## Tech Stack
- Frontend: HTML, CSS, JavaScript / React
- Backend: Node.js Express / Python Flask
- Database: MySQL / PostgreSQL
- Charts: Chart.js / Recharts
- Testing: Manual Testing, Postman
- Deployment: GitHub, Vercel, Render / Railway
